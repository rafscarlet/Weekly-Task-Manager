# Weekly Calendar Tasks App

A desktop weekly planner for managing tasks, deadlines, and schedules with offline storage.
Calendar Tasks is a simple productivity app designed to help you plan your week, organize tasks, track deadlines and well... whatever new feature comes to my mind. :P

## Features 

- [x] Weekly Calendar View 
- [x] Create, Edit, Complete, Move, Delete Tasks
- [x] Task Tags with custom colors
- [x] Task Deadline with visual indicators 
- [x] Drag & Drog to move or delete 
- [x] Copy Weekly Tasks to Clipboard
- [x] Customizable Settings 
- [x] Offline Storage (your data stays on your device)
- [x] Light/Dark Mode

## Tech Stack 

Built with: 

- Angular
- Electron
- Tailwind CSS
- Angular CDK Drag & Drop
- Typescript


## Data Storage

Your tasks, tags, and settings are stored locally on your computer using Electron's application data storage. Soon more options and control over your saved data will be added!
No account is required and no data is uploaded.

## Development 

Clone the repository: 
```bash
git clone https://github.com/rafscarlet/Weekly-Task-Manager.git
```

Install the dependencies: 
```bash 
npm install 
```

Run the development environment: 
```bash 
npm run dev  
```

Build the application: 
```bash 
npm run build 
```
